Omen File Manager

Yet another Laravel file manager

View on GitHub Download .zip Download .tar.gz

Functionalities

Responsive

This file manager will fit with any type of device, that the goal.

Resumable uploads

Uploads work in chunks so you can retry or resume it if it fails

Security in mind

This plugin includes CSP policies, CSRF protection, HTTP headers options to try to keep your project as much secure as possible. Every thing should be customizable as long as it respects the standards.

Compatible with editors

You can use this as a standalone manager, or with a wysiwyg editor such as TinyMCE or CKEditor

Image editor incuded

You can crop, resize, rotate images for a quick usage directly in the file manager

Preview your files

The manager has the ability to display, office documents, pdf, multimedia elements such as video, audio or images
